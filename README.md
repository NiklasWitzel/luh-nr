# LUH-NR
Einfaches Java App um den LUH PDF-Notenspiegel aus dem [HIS-System](http://www.his.de/) zu parsen.

##Aktuelle Version
  * 1.3.1 (30.03.2013)

##Funktionen
  * Note berechnen 
  * Zeigt Anzahl bestandenen Fächern
  * Zeigt Anzahl benoteten/unbenoteten Fächern
  * Zeigt Credit Points
